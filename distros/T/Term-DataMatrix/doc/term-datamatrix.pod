=head1 NAME

term-datamatrix - Generate terminal-based Data Matrix 2D barcodes

=head1 SYNOPSIS

B<term-datamatrix> [I<TEXT>]

I<TEXT> is the content of the Data Matrix barcode.

=head1 OPTIONS

=over 4

=item --help

Show a help message and exit.

=item --version

Show version information and exit.

=back
