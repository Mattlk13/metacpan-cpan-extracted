=head1 VERSION

=head1 DESCRIPTION

The L<Device::PaloAlto::Firewall> package has methods which make API calls against the firewall.
The return values from these methods are either hash or array references where appropriate.

Apart from some data structure re-organisation and/or simplification, the values returned are verbatim
what was receveied from the firewall in the API response.

Both the hash keys and example values are documented for each method to provide more context.
