=head1 VERSION

This document describes version 0.897 of Data::Sah::Manual (from Perl distribution Data-Sah), released on 2019-07-19.

=head1 OVERVIEW

For information about the schema language itself (syntax, types, etc) please
refer to the L<Sah> distribution. The distribution also contains some examples

L<Data::Sah::Manual::Developer>.

date): L<Data::Sah::Manual::Contributing>, L<Data::Sah::Manual::Extending>.
