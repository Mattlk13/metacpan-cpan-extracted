=head1 NAME

YAMLScript - Program in YAML

=head1 SYNOPSIS

A YAML file:

    # food.yaml
    fruit:
    - apple
    - banana
    nuts:
    - cashew
    - drupe

A YAMLScript file:

    # data.ys
    !yamlscript/v0/data

    foods:: load("food.yaml")
    numbers:: 6..9

Perl code:

    # script.pl
    use strict; use warnings;
    use YAMLScript;
    use IO::All;
    use Data::Dumper qw(Dumper);
    $Data::Dumper::Indent = 1;
    $Data::Dumper::Terse = 1;

    my $ys = io('data.ys')->all;
    my $data = YAMLScript->new->load($ys);

    print Dumper($data);

Run it:

    $ perl script.pl
    {
      'foods' => {
        'fruit' => [
          'apple',
          'banana'
        ],
        'nuts' => [
          'cashew',
          'drupe'
        ]
      },
      'numbers' => [
        6,
        7,
        8,
        9
      ]
    }

=head1 DESCRIPTION

B<YAMLScript> is a programming language that uses YAML as a base syntax.

See L<https://yamlscript.org|https://yamlscript.org> for more info.
