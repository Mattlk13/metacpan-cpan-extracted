=head1 Chess-OpeningBook

Chess-OpeningBook is a Perl library for accessing chess opening data.
See <br />
L<lib/Chess/Opening.pod> for an overview.

You can use it for reading chess opening books in <br />
LL<https://github.com/ddugovic/polyglot> format. See <br />
L<lib/Chess/Opening/Book/Polyglot.pod> <br />
for more information.

You can also lookup and browse ECO (Encyclopedia of Chess Openings)
data. See <br />
L<lib/Chess/Opening/Book/ECO.pod> <br />
for more information.
