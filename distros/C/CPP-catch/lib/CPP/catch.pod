=head1 NAME

CPP::catch - Catch test library http://catch-lib.net

=head1 SYNOPSIS

in Makefile.PL

    use XS::Install;
    WriteMakefile(
        ...
        BIN_DEPS => 'CPP::catch',
        ...
    );
    
in your C test files

    #include <catch.hpp>
    ...
    
=head1 DESCRIPTION

This module installs C<catch> headers
