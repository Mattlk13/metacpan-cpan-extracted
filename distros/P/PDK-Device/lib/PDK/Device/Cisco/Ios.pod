=head1 NAME

PDK::Device::Cisco::Ios - A Perl module for interacting with Cisco IOS devices

=head1 SYNOPSIS

    use PDK::Device::Cisco::Ios;
    my $device = PDK::Device::Cisco::Ios->new();
    # Additional functionality can be added as needed.

=head1 DESCRIPTION

The C<PDK::Device::Cisco::Ios> module is designed to handle operations specific to Cisco IOS devices. It extends the base functionality of the C<PDK::Device::Cisco> class and provides a framework that can be expanded to include more specific features as needed.

=head1 INHERITANCE

This module inherits from the C<PDK::Device::Cisco> class, which provides basic functionality for interacting with Cisco devices. The C<Ios> subclass is intended to manage specific behaviors related to Cisco's IOS operating system.

=head1 METHODS

Currently, this module does not introduce any new methods, but serves as a framework for future extensions specific to IOS devices. All methods inherited from C<PDK::Device::Cisco> can be used.
