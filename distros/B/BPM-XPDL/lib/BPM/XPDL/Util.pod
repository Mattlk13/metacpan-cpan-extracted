=head1 NAME

BPM::XPDL::Util - XPDL useful constants

=head1 INHERITANCE

 BPM::XPDL::Util
   is a Exporter

=head1 SYNOPSIS

  use BPM::XPDL::Util ':xpdl21';

Defines the namespaces used with XPDL.

=head2 Constants

Defined are C<NS_XPDL_009>, C<NS_XPDL_10>, C<NS_XPDL_20>, C<NS_XPDL_21>, and
