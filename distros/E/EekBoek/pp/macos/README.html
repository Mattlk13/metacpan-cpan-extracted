Lees dit voordat je EekBoek op MacOS gaat gebruiken!

Vanwege MacOS security constraints is het niet mogelijk de standaard file open and save dialogen te gebruiken.

De GUI Wizard heeft de beperking dat na het aanmaken van een nieuwe administratie, deze niet meteen kan worden gebruikt. Eerst moet de wizard worden afgesloten en opnieuw opgestart, waarna de nieuwe administratie kan worden gezelecteerd en geopend.

Als je zelf al eerder administraties hebt aangemaakt, dan kun je vanuit de Finder dubbelklikken op de betreffende eekboek.conf file. Ook drag and drop, Finder Open en Finder Print zijn mogelijk.

Printen vanuit de Finder is hezelfde als openen.

Hints

De EekBoek 'app' heeft twee programma's: de EekBoek GUI (ebwxshell, default) en de command line versie (ebshell). Als de dmg is gemount met defaults kan is de command line versie vanuit een Terminal window op te starten met:

/Volumes/EekBoek\ 2.04/EekBoek.app/Contents/MacOS/ebshell
