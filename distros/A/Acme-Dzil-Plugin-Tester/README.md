NAME

    Acme::Dzil::Plugin::Tester - A distro that exists so I can test various
    dzil plugins

VERSION

    version 0.04

DESCRIPTION

    It's really hard to test certain dzil plugins (especially release
    plugins) without actually doing a release. This distro lets me test
    them.
