=head1 NAME

Authen::Krb5::Principal - Kerberos 5 princpal object.

version 1.905

=head1 METHODS

=head2 C<realm()>

Returns the realm of the principal.

=head2 C<type()>

Returns the type of the principal.

=head2 C<data()>

Returns a list containing the components of the principal (everything before
the realm).
